Программа написана на python.
Файл расширения *.py является исходником.
Для запуска понадобится интерпретатор python.
В программе реализована следующая логика работы:
имитируется чёрная консоль;
вверху кнопка, нажав на которую можно открыть файл (*.txt) с исходным кодом, написанным на GUU;
пока файл не загружен, ничего выводиться толкового не будет;
по нажатии на кнопку и выборе файла последний считается, и выведется в консоли надпись PROGRAMM: "путь к считанному файлу";
далее от пользователя ожидается ввод одной из команд: i, o, trace, var;
выполнение функции main происходит соответственно следующим образом:
i - выполняется main-функция со всеми объявленными функциями внутри main;
o - выполняется main-функция без выполнения объявленных внутри main функций;
trace - происходит печать с номерами строк всех команд внутри main;
var - печать переменных с их значениями;
если нажать клавиши вверх или влево и выйти за пределы строки, чтобы вернуться, необходимо нажимать противоположные клавиши вниз/вправо, чтобы вернуться к строке ввода.
